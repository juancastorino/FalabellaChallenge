## Falabella Challenge.

- Prints all the numbers from 1 to 100
- Only 1 If
- You can't use `else`, `else if` or ternary
- Unit tests
- Feel free to apply your SOLID knowledge

## Stack

- PHP 7.4+
- PhpUnit 9
- Boostrap 5
